Class Storm

Storm represents information about registered lightnings with a specified radius of monitoring covered by the given point

Storm has properties describing: - number (number of cloud-to-ground lightning in specified radius from a selected point) - distance (distance to the nearest registered lightning in km) - direction (direction to the nearest lightning [E, E, N, NW, W, SW, S, SE]) - period (number of minutes of time period containing the data [10, 15, 20 minutes]) - point (selected point) - radius (radius covered by Point in km)
